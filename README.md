# أكاديمية المصفوفة - موقع استقطاب المستثمرين

موقع ويب احترافي مخصص لاستقطاب المستثمرين في قطاع الذكاء الاصطناعي والتقنية المتقدمة والحوسبة السحابية والتحول الرقمي.

## 🌟 المميزات

- **تصميم حديث واحترافي**: واجهة مستخدم رائعة وسهلة الاستخدام
- **استجابة ديناميكية**: يعمل بشكل مثالي على جميع الأجهزة (هواتف، أجهزة لوحية، أجهزة كمبيوتر)
- **وسائل التواصل الاجتماعي**: روابط مباشرة إلى جميع منصات التواصل الاجتماعي
- **نماذج اتصال متقدمة**: تحقق من البيانات والتحقق من صحتها
- **محسّن لمحركات البحث**: SEO friendly
- **أداء عالي**: تحميل سريع وسلس

## 📋 الصفحات

1. **الرئيسية (index.html)**: صفحة الهبوط الرئيسية مع ملخص الخدمات والإحصائيات
2. **عن الشركة (pages/about.html)**: معلومات عن أكاديمية المصفوفة والفريق والقيم
3. **الخدمات (pages/services.html)**: وصف مفصل للخدمات المقدمة والحزم البرامجية
4. **فرص الاستثمار (pages/opportunities.html)**: معلومات عن الفرص الاستثمارية المتاحة
5. **اتصل بنا (pages/contact.html)**: نموذج اتصال وتفاصيل الاتصال والأسئلة الشائعة
